03 // LIVE_CODING_TERMINAL

Implementasi Flask & GeoPandas

1

Environment Setup

Install dependensi utama untuk web mapping.

$ pip install geopandas folium flask
2

Reading Geospatial Data

Load Shapefile dan konversi Coordinate Reference System (CRS) ke WGS84.

import geopandas as gpd
gdf = gpd.read_file('peta_indonesia.shp')
# Align CRS with Folium (EPSG:4326)
gdf = gdf.to_crs(epsg=4326)
3

Flask Routing

Membuat endpoint untuk mengirimkan peta ke frontend.

@app.route('/map')
def tampilkan_peta():
m = folium.Map(location=[-6.2, 106.8], zoom_start=10)
folium.GeoJson(gdf).add_to(m)
return render_template('index.html', peta_interaktif=m._repr_html_())
4

HTML Template Injection

Merender output HTML Folium menggunakan Jinja2 safe filter.

<!-- templates/index.html -->
<div id="map-container">
{{ peta_interaktif|safe }}
</div>
